ダンジョンタイル詰め合わせ

アリアンロッドを想定して作成したランダムダンジョン用タイルの詰め合わせです。
（アリアンロッド上級ルール掲載のランダムダンジョンとの互換性はありません）
１枚のタイルは10×10マスを想定していますが、20×20マスに引き延ばせばD&Dでも使える気がします。
なお、おまけとして大型タイルが二枚入っていますが、こちらは20×20マスを想定して作ってあります。

ランダムダンジョンフォーマット.jpgを表示すると、どういうコンセプトで作成されているのかわかる……かもしれません。
ひたすらつなぎ合わせて巨大なマップに挑むのもよいかと思います。

本ファイルを含むrdd1.zipのすべてのファイルは自由に利用・改変が可能です。
連絡も必要ありません。

なお、このファイルを利用して生じたいかなるトラブルに対しても、作者である超小兎は責任を負いません。
